ARGUS - Cleanup Multi-Tenant

Strumento per rimuovere device "estranei" dalla pagina di un cliente. Funziona direttamente nel browser, zero installazione.

1

Apri il tuo Center e logga come admin

Vai sul tuo Center e fai login normalmente:

https://argus.86bit.it

2

Trascina questo pulsante nei tuoi PREFERITI del browser

Letteralmente: clicca sul pulsante arancione qui sotto, tienilo premuto, e trascinalo dentro la barra dei preferiti del browser (in alto). Si salvera' come segnalibro.

ARGUS Cleanup

Nota: se la barra dei preferiti non e' visibile, premi Ctrl+Shift+B (Windows) per mostrarla.
3

Vai sulla pagina del cliente con device sporchi

Esempio: pagina del cliente Galvan. Devi essere DENTRO la pagina, l'URL deve essere tipo:

https://argus.86bit.it/clients/90f83b9b-10f3-477d-b9f5-d1ca864e4199

(o qualunque sia il path nel tuo Center)

4

Clicca il segnalibro "ARGUS Cleanup" che hai appena salvato

Lo script:

  • Recupera il client_id dall'URL automaticamente (se serve te lo chiede)
  • Ti chiede la subnet "naturale" del cliente (es. 10.100.61.0/24 per Galvan)
  • Lista i device "estranei" (IP fuori dalla subnet + Auto-127.0.0.1)
  • Ti mostra l'elenco e chiede conferma
  • Se confermi -> DELETE via API + ricarica la pagina
Risultato: dopo il reload vedrai SOLO i 5 device legittimi di Galvan (Switch01/02 HP, NAS Backup, UPS XANTO 3000, iLO GALVANSRV).

In caso di problemi

  • "Devi essere loggato" -> rifai login sul Center prima di cliccare il bookmarklet
  • "Client ID del cliente" chiesto via prompt -> incolla l'UUID del cliente (lo trovi nell'URL o nella tabella Clienti)
  • "Subnet" -> per Galvan e' 10.100.61.0/24. Per altri clienti adatta.
  • Errore HTTP 401 -> il token JWT e' scaduto, fai logout/login
